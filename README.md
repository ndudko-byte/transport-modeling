# Транспортное моделирование — Лендинг

B2B лендинг для компании транспортного моделирования (PTV Vissim).

## Что это

Продающий сайт под SEO-продвижение и лидогенерацию. Целевая аудитория — застройщики, проектировщики, администрации городов и крупный бизнес, которым нужна экспертиза по транспортным потокам.

## Функциональность

- Блоки услуг, проектов, команды
- Форма заявки (интеграция Bitrix24)
- Чат (Bitrix24 Open Lines)
- Калькулятор стоимости (JS)
- Блог / полезные статьи (SEO)

## Технологии

- HTML5 / CSS3 / Vanilla JS
- Bitrix24 (CRM, чат)
- Яндекс.Метрика

## Как запустить

Открыть `index.html` в браузере или использовать Live Server в VS Code.

## Статус

В разработке — быстрый запуск и SEO-продвижение.
