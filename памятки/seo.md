## Технический SEO (чеклист перед запуском)

### Мета-теги (index.html)
- [ ] `<title>` — до 60 символов, ключевое слово в начале
  - Пример: `Транспортное моделирование PTV Vissim — заказать в Москве`
- [ ] `<meta name="description">` — до 160 символов, призыв к действию
- [ ] `<meta name="keywords">` — 5-10 ключевых слов (не переспамить)
- [ ] `<link rel="canonical">` — указывает на основной URL страницы
- [ ] Open Graph теги (`og:title`, `og:description`, `og:image`) — для шаринга

### Структура страницы
- [ ] Один `<h1>` на страницу
- [ ] Правильная иерархия h1 → h2 → h3
- [ ] Семантические теги: `<main>`, `<section>`, `<article>`, `<nav>`, `<footer>`
- [ ] Alt-теги на всех изображениях

### Файлы
- [ ] `sitemap.xml` — перечень всех страниц, отправить в Яндекс.Вебмастер
- [ ] `robots.txt` — разрешить индексацию нужных страниц

### Schema.org (структурированные данные)
- [ ] Organization — название, адрес, телефон, email
- [ ] Service — описание каждой услуги
- [ ] LocalBusiness — если есть физический офис
- [ ] Article — для каждой статьи блога
- [ ] FAQ — если будет блок вопрос/ответ

### Скорость
- [ ] Сжать изображения (TinyPNG или аналог)
- [ ] Минифицировать CSS и JS перед деплоем (опционально)
- [ ] Проверить PageSpeed Insights — цель 90+
