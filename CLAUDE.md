# Транспортное моделирование — Лендинг

## О проекте

B2B продающий лендинг для компании транспортного моделирования (PTV Vissim).
Целевая аудитория: застройщики, проектировщики, администрации городов, крупный бизнес.
Цель: лиды через форму заявки, калькулятор стоимости, чат. SEO-продвижение + блог со статьями.

**Тип:** Статичный лендинг (HTML/CSS/JS), возможна миграция на бэкенд в будущем
**Для кого:** Рабочий продукт, новая услуга, быстрый запуск
**Технологии:** HTML5 / CSS3 / Vanilla JS
**Интеграции:** Bitrix24 (заявки, чат Open Lines), Яндекс.Метрика, Schema.org

## SEO-правила (соблюдать всегда)

- Каждая страница: уникальный `<title>` (до 60 символов) и `<meta description>` (до 160 символов)
- Один `<h1>` на страницу, иерархия заголовков h1→h2→h3
- Все изображения: атрибут `alt` с описанием
- Семантические теги: `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`
- Schema.org разметка: Organization, Service, LocalBusiness — добавить в `<head>`
- Ссылки между статьями блога — внутренняя перелинковка
- Файлы: `sitemap.xml` и `robots.txt` — создать при запуске

## Интеграции

### Bitrix24
- Формы заявок: через Bitrix24 Web Form (embed) или REST webhook (URL в .env)
- Чат: виджет Bitrix24 Open Lines — один `<script>` в `<body>`
- Калькулятор стоимости: чистый JS, результат отправляется вместе с формой заявки

### Аналитика
- Яндекс.Метрика: счётчик + цели (отправка формы, клик по телефону, открытие чата)
- Google Analytics 4: дополнительно (особенно если будет платная реклама)
- UTM-метки: для всех рекламных кампаний

## Важные решения

- Бэкенд не нужен сейчас — все интеграции через embed/webhook
- Калькулятор — JS без сервера, считает на стороне клиента
- Блог — статические HTML-страницы (легко, быстро, хорошо для SEO)
- В будущем возможна миграция на полноценный сайт с бэкендом
